# HC:// TRUST LAYER v0.1.0 Release Notes

Date: 2026-05-19
Status: Release Candidate (Experimental)

## Release Summary

v0.1.0 establishes the first public release-candidate documentation baseline for HC:// TRUST LAYER.

This version includes:

- protected PR workflow documentation;
- CodeQL and Dependabot integration documentation;
- validation pipeline documentation;
- real record validation guidance;
- safer normalization flow notes;
- QR/witness/trust documentation;
- visible verification demo flow;
- limitations and trust model clarifications;
- architecture roadmap and glossary references.

## Important Trust and Security Notes

- **v0.1.0 is experimental.**
- **HC:// verifies integrity and provenance signals, not objective truth.**
- **Witness and trust scoring remain experimental.**
- **No production security guarantees are provided yet.**

## What Verification Means in v0.1.0

In this release, verification indicates that record-linked technical signals (such as hash consistency, schema conformance, and traceable provenance context) are reproducible under the documented process.

Verification in v0.1.0 does **not** claim that a record's narrative content is factually correct in an absolute sense.
