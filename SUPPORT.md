# Support

Thank you for using HC:// TRUST LAYER.

## How to ask questions
For usage questions, implementation guidance, or contribution help, please open a GitHub Discussion or a GitHub Issue in this repository and include:
- What you are trying to achieve.
- What files, records, or commands are involved.
- What you expected and what happened instead.

## Reporting security concerns
Please do not disclose sensitive vulnerabilities in public issues.
Follow the responsible disclosure instructions in `docs/SECURITY.md` when reporting security concerns.

## Experimental status
HC:// TRUST LAYER is experimental. Behaviors, verification flows, and trust guidance may evolve as the project matures.
